Starting a Node

This dapp requires a node to be running. You may find a puplically available node such as rnode-test.rhobot.net useful. Or you may consider the following and start your own local node.

  • Required: Node must be a validator. Private keys are available in ./rnode/genesis
  • Required: Node must have non-zero weight. Using an above key ensures this.
  • Suggestion: Standalone nodes are easier to start for testing purposes.

This example command will work in most cases.
rnode run -s -n --validator-private-key bc6391d3acdd9135dde2a0eafdd2f4d9edb10111efff5d72563b10a69e936535

Register for Status

Enter a name that you want to go by. This name is the channel on which you will send commands to update and check your status. This is a huge security hole, and in a future release, we will use unforgeable names, but RNode does not have a working name registry yet.
